# Marketing Campaign Performance Analytics

## Project Overview

This project analyzes marketing campaign performance across different channels, campaign types, customer segments, and target audiences. The objective is to evaluate campaign effectiveness and identify patterns related to ROI, engagement, conversion rate, and acquisition cost.

## Tools & Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- SQLite
- Jupyter Notebook
- Power BI

## Dataset

Marketing Campaign Performance Dataset
https://www.kaggle.com/datasets/manishabhatt22/marketing-campaign-performance-dataset
